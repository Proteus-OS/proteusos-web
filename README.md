# 🌐 ProteusOS Web Source

Este repositorio contiene el código fuente del sitio oficial de **ProteusOS**.

## 🚀 Características
- Sitio estático optimizado.
- **Soporte Multi-idioma:** Versiones completas en Español e Inglés.
- **Diseño Responsivo:** Adaptado para dispositivos móviles y tablets.
- Desplegado automáticamente mediante **GitHub Pages**.

## 📂 Estructura del Proyecto
- `/index.html`: Página principal (Español).
- `/en/index.html`: Versión internacional (Inglés).
- `/assets/`: Logotipos, iconos y tipografías.
- `/styles.css`: Hoja de estilos compartida.
