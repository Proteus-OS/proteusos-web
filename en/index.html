ProteusOS v1.0 - Aegis

ProteusOS is a distribution based on Debian 13 (XFCE), optimized for cybersecurity analysts. It provides a pre-configured ecosystem of tools for threat detection, log management, and incident response in a hardened environment.

Deployment Profiles

The system features a TUI installer that allows you to select three protection levels based on computing power and monitoring scope:

ProteusOS LITE

Endpoint and basic perimeter protection. Optimized native installation.

  • Wazuh: Unified SIEM/XDR.
  • CrowdSec: Collaborative IPS.
  • AdGuard Home: DNS Sinkhole.
  • Tailscale: Secure Mesh Network.

ProteusOS STANDARD

Network traffic inspection and security process automation.

  • Everything in LITE +
  • Suricata: Network IDS/IPS.
  • Shuffle: SOAR Automation.
  • Active Packet Inspection.

ProteusOS FULL

Total forensic analysis and centralized threat intelligence.

  • Everything in STANDARD +
  • Zeek: Protocol Analysis.
  • TheHive & Cortex: Incident Mgmt.
  • Full Forensic Suite.

System Architecture

ProteusOS utilizes a hybrid deployment matrix to maximize isolation and performance:

  • Native Installation (Systemd): System core, network monitors (Suricata/Zeek), and security agents.
  • Isolated Containers (Docker): Management and orchestration apps (Shuffle, TheHive, Cortex) for easy updates and scalability.
  • GUI: XFCE tuned for low resource consumption.

Hardware Requirements

Minimum (LITE)

  • CPU: 2 Cores / RAM: 4 GB
  • Disk: 40 GB SSD

Optimal (STANDARD/FULL)

  • CPU: 4+ Cores / RAM: 16 GB
  • Disk: 100 GB+ SSD
