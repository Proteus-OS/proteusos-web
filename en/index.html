ProteusOS v1.0 - Aegis

ProteusOS is a distribution based on Debian 13 (XFCE), optimized for cybersecurity analysts. It provides a hardened ecosystem for threat detection and incident response without compromising performance.

⬇ Download ISO (v1.0) GitHub Organization ↗

Deployment Profiles

ProteusOS LITE

Endpoint and perimeter protection. Optimized native installation.

  • Wazuh: Unified SIEM/XDR.
  • CrowdSec: Collaborative IPS.
  • AdGuard Home: DNS Sinkhole.
  • Network Tools (Nmap/Tcpdump).

ProteusOS STANDARD

Network traffic inspection and security process automation.

  • Everything in LITE +
  • Suricata: Network IDS/IPS.
  • Shuffle: SOAR Automation.

ProteusOS FULL

Total forensic analysis and centralized threat intelligence.

  • Everything in STANDARD +
  • Zeek: Protocol Analysis.
  • TheHive & Cortex: Incident Management.

Hybrid Architecture

ProteusOS maximizes efficiency through a dual deployment matrix:

  • Native (Systemd): System core and critical network monitors for zero latency.
  • Containers (Docker): Isolated orchestrators and management apps for increased stability.

Hardware Requirements

Minimum (LITE)

  • CPU: 2 Cores
  • RAM: 4 GB
  • Disk: 40 GB SSD

Optimal (FULL)

  • CPU: 4+ Cores
  • RAM: 16 GB
  • Disk: 100 GB+ SSD
